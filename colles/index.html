Programmes de colles

Rendez vous sur la page d'abonnements de votre calendrier.

Pour Google rendez vous sur le lien: https://calendar.google.com/calendar/u/0/r/settings/addbyurl

Suivez cette démarche :

  • Copiez-collez l'url correspondant à votre groupe de colle.
  • Si le colloscope est modifié, les changements seront automatiques sous 24h (délais imposé par Google).
  • Si vous n'y parvenez pas, visitez https://support.google.com/calendar/answer/37100.
